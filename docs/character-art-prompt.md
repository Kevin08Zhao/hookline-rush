# Comet Courier character prompt

This prompt is intended for concept exploration. The in-game asset should still be rebuilt as
separate puppet parts so that running, jumping, prone crawling, grappling, slamming, and
knockback can share the existing animation rig.

```text
Create an original 2D cartoon game character concept sheet for a neon sci-fi grappling
parkour game.

Character: “Comet Courier”, a compact athletic and androgynous young courier. Give the
character an oversized rounded meteor-shaped helmet with a small swept-back coral fin, one
luminous visor slit, a short asymmetrical navy jacket, one off-white armored shoulder, a
compact grappling launcher integrated into the front forearm, a utility belt with a triangular
comet badge, tapered jogger pants, oversized split-sole parkour sneakers, and two short solid
energy-scarf ribbons trailing behind the neck. The silhouette should feel friendly, brave,
slightly mischievous, and instantly readable at 48–64 pixels tall.

Style: polished original 2D cartoon game art, bold clean shapes, strong dark outer contour,
subtle cel shading, premium indie-game finish, not pixel art, no resemblance to an existing
franchise. Palette: deep navy bodysuit, luminous cyan primary accent, coral-orange secondary
accent, off-white helmet and armor, very small violet details.

Show exactly three consistent side-view poses of the same character, all facing right:
1. neutral standing pose with every limb clearly separated;
2. dynamic running pose;
3. low prone-crawl pose.

Also show a clean exploded-parts row containing: helmet/head, torso, front upper arm, front
forearm with grapple device, back upper arm, back forearm, front thigh, front shin/shoe, back
thigh, back shin/shoe, upper scarf ribbon, and lower scarf ribbon. Every limb must have thick
overlap cuffs at the shoulder, elbow, hip, and knee so independently rotated parts never reveal
gaps. Keep identical proportions, costume details, lighting, and colors across all views.

Use a perfectly flat solid #00ff00 chroma-key background. No shadow, gradient, floor,
environment, text, labels, logo, watermark, extra character, extra limb, loose hair, cape,
long coat, smoke, transparent material, extreme perspective, or overlapping limbs in the
neutral pose. Keep all pieces fully visible with generous spacing and crisp opaque edges.
```

For alternate characters, preserve the exact part list, side-view proportions, overlap zones,
and small-scale readability. Change only the silhouette theme, helmet, torso costume, forearm
device casing, shoes, scarf shape, and palette.
